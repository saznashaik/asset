# **App Name**: QuickSight Executive AI

## Core Features:

- AWS QuickSight Integration: Display dashboards and reports from AWS QuickSight directly within the app.
- Top Chat Agent Integration: Integrate data from the top chat agent to provide insights into customer interactions.
- AI-Powered Questioning: Allow users to ask questions about the data, using AI to interpret the question and fetch the relevant information. Use the AI tool to decide if relevant context is missing and to look up external documentation to enhance the query.  
- Executive Dashboard: Display a customizable executive dashboard with key metrics and visualizations.
- Data Refresh: Enable automatic or manual data refresh from connected data sources.
- User Authentication: Secure the application with user authentication.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) for a professional and trustworthy feel.
- Background color: Light gray (#F9FAFB) to provide a clean and modern canvas.
- Accent color: Orange (#EA580C) to highlight important elements and calls to action.
- Headline font: 'Inter', sans-serif, for clear and modern headings.
- Body font: 'Inter', sans-serif, to maintain consistency and readability.
- Use consistent and clear icons to represent data and actions.
- Design a clear and intuitive layout with well-defined sections for easy navigation.
- Implement subtle transitions and animations to enhance user experience without being distracting.
